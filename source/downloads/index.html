Installing from source

For most people the easiest way now to try out the latest Christmas Perl 6 (6.c language version) is to build Rakudo Star from an extracted source tarball.

You will need recent versions of perl, git, make and gcc. Also a basic familiarity with UNIX.

Windows users will be able to directly install a MSI from the download directory below shortly

For full instructions see http://rakudo.org/how-to-get-rakudo/

A quick start for Linux/Mac/BSD etc. follows:

        % wget http://rakudo.org/downloads/star/rakudo-star-2016.01.tar.gz
        % tar xfz rakudo-star-2016.01.tar.gz 
        % cd rakudo-star-2016.01
        # perl Configure.pl --gen-moar --prefix /opt/rakudo-star-2016.01     
        # make install

Developers interested in tracking the latest versions to help with Perl 6 itself are recommended to consider rakudobrew.

For full rakudobrew instructions see https://github.com/tadzik/rakudobrew
