Download Raku

Raku™ is a language specification similar to C and C++. To actually run programs written in Raku one needs a Raku implementation, such as Rakudo™.

Rakudo™

Rakudo™ is the most actively developed and mature implementation of the Raku Programming Language. Rakudo can be downloaded from its website.
