Installing from source

Currently the best way to install the Christmas Perl 6 release is to build from source either via an extracted tarball or rakudobrew (which uses git).

For full instructions including how to build from a tarball see http://rakudo.org/how-to-get-rakudo/

For convenience of path management and Perl 6 updates the use of rakudobrew is recommended.

For full rakudobrew instructions see https://github.com/tadzik/rakudobrew

You will need recent versions of perl, git, make and gcc.

A quick start for Linux/Mac rakudobrew follows: 

git clone https://github.com/tadzik/rakudobrew ~/.rakudobrew 
export PATH=~/.rakudobrew/bin:$PATH 
rakudobrew build moar 2015.12
rakudobrew build-panda

Note: if you have old installation of rakudobrew or Perl 6 and you're having issues updating, you may try deleting ~/.rakudobrew and ~/.perl6 directories first and starting from scratch.
